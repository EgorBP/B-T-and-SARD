### Страница: Главная (`/`)

**Тип доступа:** публичный
**Основные блоки:**

1. Шапка: логотип, меню (Главная, Публичные треки, Войти), аватар (если авторизован)
2. Блок приветствия: краткое описание сайта, кнопка "Перейти к трекам"
3. Футер: контакты, ссылки на соцсети

**Динамический контент:** отображение имени пользователя при авторизации

---

### Страница: Публичные треки (`/tracks`)

**Тип доступа:** публичный
**Основные блоки:**

1. Шапка: логотип, меню (Главная, Публичные треки, Войти), аватар (если авторизован)
2. Список треков: карточки треков с названием, автором, кнопкой воспроизведения, типом приватности

**Динамический контент:** карточки треков формируются из базы данных, при прослушивании трека появляется плеер

---

### Страница: Детальная страница трека (`/track/:id`)

**Тип доступа:** публичный
**Основные блоки:**

1. Шапка: логотип, меню, аватар
2. Информация о треке: название, автор, длительност, приватность

**Динамический контент:** данные трека подгружаются из базы

---

### Страница: Аутентификация (`/auth/*`)

**Тип доступа:** публичный
**Основные блоки:**

1. Шапка: логотип
2. Форма входа/регистрации: поля username, password, кнопка "Продолжить"
3. Ссылка на восстановление пароля

**Динамический контент:** форма отправляет данные на сервер для проверки

---

### Страница: Личный кабинет (`/profile`)

**Тип доступа:** приватный
**Основные блоки:**

1. Шапка: логотип, меню (Главная, Публичные треки, Мой профиль), аватар
2. Секция «Мои треки»: список загруженных треков с кнопками управления (удаление, смена приватности)
3. Секция действий: кнопка "Загрузить новый трек", кнопка "Изменить профиль"

**Динамический контент:** данные загруженных треков пользователя подгружаются из базы

---

### Страница: Изменение профиля (`/profile/change`)

**Тип доступа:** приватный
**Основные блоки:**

1. Шапка: логотип, меню, аватар
2. Форма изменения профиля: поля для изменения username, пароля
3. Кнопка сохранения изменений

**Динамический контент:** форма отправляет данные на сервер для обновления

---

### Страница: Загрузка трека (`/upload`)

**Тип доступа:** приватный
**Основные блоки:**

1. Шапка: логотип, меню, аватар
2. Форма загрузки: поле для названия трека, выбор файла, кнопка "Загрузить"
3. Список загруженных треков с кнопками управления приватностью

**Динамический контент:** список треков обновляется после загрузки нового файла

---

### Страница: Поиск треков (`/search`)

**Тип доступа:** публичный
**Основные блоки:**

1. Шапка: логотип, меню, аватар
2. Поле поиска: ввод запроса, кнопка поиска
3. Секция результатов: карточки треков с названием и автором

**Динамический контент:** результаты поиска подгружаются из базы данных по введённому запросу
