**SpotX**

**Описание проекта:**

SpotX — это динамическое веб-приложение для публикации и прослушивания аудиотреков. Пользователи могут регистрироваться в системе, загружать свои аудиофайлы и управлять их доступностью. Приложение позволяет делиться треками с другими пользователями или оставлять их приватными для личного использования.

Система разделена на публичную и приватную части. Публичная часть позволяет просматривать и прослушивать доступные треки, а приватная часть предоставляет пользователю доступ к личному кабинету, где можно управлять своими загрузками. Архитектура приложения построена по клиент-серверной модели: серверная часть обрабатывает запросы, управляет базой данных и хранением файлов, а клиентская часть отвечает за отображение интерфейса и взаимодействие с пользователем.


**Целевая аудитория:**

* Пользователи, которые хотят хранить и прослушивать свои аудиотреки
* Музыканты или авторы, желающие делиться своими записями
* Пользователи, ищущие новые аудиотреки для прослушивания


**Ключевой функционал:**

* Регистрация пользователей
* Авторизация и управление пользовательской сессией
* Загрузка аудиотреков
* Прослушивание публичных треков
* Управление приватностью треков
* Просмотр списка собственных треков в личном кабинете
* Доступ к медиафайлам через сервер приложения
